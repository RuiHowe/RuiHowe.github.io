hello world和classpath

错误
pubilc class hello
{ public static void main(String[] args)
{
System.out.println(hello world);
} }

结果：
hello.java:1: 错误: 需要class, interface或enum
pubilc class hello
^ hello.java:5: 错误: 需要’)’
System.out.println(hello world);
^
hello.java:5: 错误: 非法的表达式开始

classpath
class path是 “类环境变量” path是一些exe啊之类的
set classpath=c:\myclass 会在c:\myclass
而 set classpath=c:\myclass； 在c:\myclass找，之后会在原本的找
比如 F:\java demo>set classptah=c:\myclass

F:\java demo>set classptah=.;%classpath% .是“当前目录”的意思。
